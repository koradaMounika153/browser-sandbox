# Browser-Based Coding Sandbox

A browser-based coding sandbox built using the MERN stack that allows users to create, edit, delete, and manage multiple files directly in the browser with real-time live preview support.

## Features

- Multi-file browser IDE
- Monaco code editor integration
- Live React preview using Sandpack
- Create and delete files dynamically
- Real-time code updates
- MongoDB project persistence
- Session persistence across refresh
- MERN stack architecture

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Monaco Editor
- Sandpack

### Backend
- Node.js
- Express.js
- MongoDB Atlas

## Installation

### Frontend

npm install
npm run dev

### Backend

cd server
npm install
npm run dev
